# MatterVis Scripts

The numbered scripts are runnable examples for the public API:

- `01_quick_render.py`: CIF to Plotly PNG/HTML.
- `02_coordination_analysis.py`: topology analysis and score dump.
- `03_display_modes_panel.py`: formula/unit-cell/shell comparison.
- `04_static_publication.py`: matplotlib publication PNG/PDF.
- `05_app_and_api.py`: launch the Dash app and drive REST endpoints.
- `06_cp2k_cube_orbital.py`: cube/orbital isosurface rendering.
- `07_scene_tabs.py`: multi-scene/tab API example.
- `08_ortep_interactive.py`, `09_ortep_publication.py`: ORTEP examples.

Shared sample CIFs live in `data/`. Generated outputs land in `_outputs/`. Local or unpublished analysis belongs under `private/`, which is intentionally ignored.
